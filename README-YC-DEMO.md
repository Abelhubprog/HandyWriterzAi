## 🌟 What is HandyWriterz?

HandyWriterz is a revolutionary AI-powered academic writing platform that combines **30+ specialized agents** to transform complex research materials into publication-ready content. From PhD dissertations to market research reports, our system handles multimodal inputs (PDF, DOCX, audio, video) and delivers sophisticated academic output with proper citations and analysis.

### 🎯 Perfect for YC Demo Day

This system showcases the power of **multiagent orchestration** in solving complex, real-world problems. Watch as our agents collaborate in real-time to:
- Process 10+ different file types simultaneously
- Generate 20,000+ word dissertations with proper academic structure
- Maintain citation integrity and academic standards
- Provide real-time cost tracking and quality metrics

## 🚀 Quick Start

### Prerequisites
- Docker and Docker Compose
- Python 3.11+
- Node.js 18+
- API Keys: OpenAI, Google (Gemini), Anthropic (Claude)

### 1. Clone and Setup
```bash
git clone https://github.com/your-org/multiagentwriterz.git
cd multiagentwriterz
cp .env.production.example .env
```

### 2. Configure API Keys
Edit `.env` file with your actual API keys:
```bash
OPENAI_API_KEY=sk-your-openai-key-here
GOOGLE_API_KEY=your-google-gemini-key
ANTHROPIC_API_KEY=your-anthropic-key
```

### 3. Start Development Environment
```bash
# Quick development setup
./start-dev.sh
```

### 4. Start Production Environment
```bash
# Full production deployment
./start-production.sh
```

### 5. Run YC Demo Test
```bash
# Comprehensive dissertation test
python test-dissertation-journey.py
```
